HTML Page Structure

Here is the scheme of page structure

  • .off-screen-content - scrollable content visible after navigation button is clicked
  • .off-screen-navigation - navigation for .off-screen-content
  • .outer-wrapper - main wrapper for photos
  • .navigation - navigation button

Creating and Editing Slides

Code for one slide with image and description is this

<div class="slide" id="8"> // Every .slide must have a unique ID
    <div class="description">
        <h2 class="animate">Dark Dream </h2>
        <dl class="animate">
            <dt>Client: </dt>
            <dd>Personal Work </dd>
            <dt>Photographer: </dt>
            <dd>Jane Sane </dd>
            <dt>Category: </dt>
            <dd>Portrait </dd>
         </dl>
         <div class="additional-info animate">
             <dl>
                 <dt>ISO:</dt>
                 <dd>700</dd>
                 <dt>Shutter Speed:</dt>
                 <dd>1/160s</dd>
                 <dt>Focal Length:</dt>
                 <dd>38mm</dd>
            </dl>
        </div>
    </div>
    <div class="image" data-background="assets/img/8.jpg"></div> // Path to image
</div>

Note: animate class creates fade in animation of an element after slide animation.

Map Settings

Necessary variables

To get map to work properly, you need to set these variables showed lower. This script is at the bottom of the homepage HTML file.

var _latitude = 48.47292127; // Latitude of the map center
var _longitude = 4.28672791; // Longitude of the map center
var element = "map-item"; // ID of the map element. In this case <div class="map" id="map-item"></div>
var markerImage = "assets/img/map-marker.png"; // icon for marker
var mapTheme = "light"; // color theme for map. Use "light" or "dark"

Google Maps API Key

In case the map is not displayed, or there will be only grey background with "Oops! Something went wrong" message, you probably don't have an API key.

Get your API Key here:

https://developers.google.com/maps/documentation/javascript/get-api-key#get-an-api-key

At the bottom of the page with map find:

<script type="text/javascript" src="http://maps.google.com/maps/api/js"></script>

Add your API key after the /js
